# Bookish Final Presentation

This folder contains the materials for the **final project presentation** of **Bookish**, a platform designed to help readers discover book communities, track reading, and receive personalized recommendations.

## Contents
- Presentation slides
- Demo resources

A live walkthrough of the platform is shown in the demo video:

**Demo Link:** https://drive.google.com/file/d/1UA0MJa7v9j6DMzDtYYou7BqiT5ZeTbzF/view?usp=sharing

The demo highlights:
- Discovering book clubs and reading events
- Participating in discussions
- Managing a personal reading library
- Receiving personalized book recommendations
