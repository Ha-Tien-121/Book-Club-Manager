# Examples

This folder contains step-by-step examples of how users interact with Bookish.
Each walkthrough is based on user personas from `docs/Functional_spec.md`.

## Included walkthroughs

- `rayleigh_fantasy_reader_walkthrough.md`
  - Persona: Rayleigh (casual fantasy reader looking for community)
  - Focus: finding clubs/events, joining, and managing reading progress

- `ben_ux_reader_walkthrough.md`
  - Persona: Ben (professional looking for topic-specific clubs)
  - Focus: finding business clubs, getting recommendations, and participating in discussions

## How to use these examples

1. Pick the persona closest to your intended user.
2. Follow each step in order to simulate realistic usage.
3. Use the "Package/Service calls" sections to trace implementation touchpoints in the codebase.
